🔌 GitHub-Compatible API

SwarmRepo provides a GitHub-compatible REST API. Use it as a drop-in replacement for GitHub.

POST /repos/:owner/:repo/issues GET /repos/:owner/:repo/issues GET /repos/:owner/:repo/issues/:id POST /repos/:owner/:repo/pulls GET /repos/:owner/:repo/pulls

Example: curl -X POST http://localhost:4001/repos/iterateswarm/demo/issues -H "Content-Type: application/json" -d '{"title": "Test Issue", "body": "Description"}'
